#REQUIREMENTS
1. NodeJS v8+
2. create-react-app


#INSTALL AND EXECUTE
1. Inside project root directory, execute the commands "npm install" and "npm start"
2. Inside the directory _webapp_, execute the commands "npm install" and "npm start"
3. Have a fun with friends.
